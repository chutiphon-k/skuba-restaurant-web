## Package
- React
- Webpack 2
- Babel
- Bulma
- Expressjs
- Socket.io
- MQTT

## Using Web
```bash
git clone https://github.com/chutiphon-k/skuba-restaurant-web.git
cd web
npm install
npm run dev # For run development
npm run build # For run prodtion
```

## Using Server
```bash
git clone https://github.com/chutiphon-k/skuba-restaurant-web.git
cd server
npm install
npm run dev # For run development
```
